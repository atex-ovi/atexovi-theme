# Security Policy for Atexovi Theme

## Supported Versions

| Version | Supported          |
| ------- | ----------------- |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

> [!NOTE]
> Version 1.x was a trial/beta release. Users are encouraged to upgrade to 2.x for the latest fixes and improvements.

## Reporting a Vulnerability

If you discover a security issue in Atexovi Theme, please report it **privately** via the following channels:

[![Telegram](https://img.shields.io/badge/Telegram-29A9EB?style=for-the-badge&logo=telegram&logoColor=white)](https://t.me/atexovi)  
[![Messenger](https://img.shields.io/badge/Messenger-00B2FF?style=for-the-badge&logo=messenger&logoColor=white)](https://m.me/atex.ovi)

Include as much detail as possible:

- Steps to reproduce  
- Version of Atexovi Theme  
- Termux / Android environment  

### Response

- Issues will be acknowledged and discussed privately.  
- Fixes will be released in the next update.  

> **Important:** Do **not** disclose security issues publicly until a fix has been released.
